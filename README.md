# Projet Chimera

## Vue d'ensemble

Le Projet Chimera est une plateforme de simulation distribuée conçue pour modéliser des interactions complexes entre agents autonomes. Ce projet a été développé dans le cadre de nos études universitaires en systèmes distribués.

## Fonctionnalités

* Gestion d'agents dynamiques
* Communication inter-processus sécurisée
* Interface de visualisation des simulations (version CLI)

## Technologies

* Python 3.x
* Framework de communication ZeroMQ
* Docker

## Installation & Utilisation

Voir `src/main.py` pour le point d'entrée.
